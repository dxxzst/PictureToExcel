<h3>将图片各个像素转换为Excel中的单元格背景色<h3/><br/>
1. 读取图片中每个像素RGB值<br/>
2. 创建Excel文档<br/>
3. 将RBG值依此设置为单元格背景色<br/>
4. 单元格形成马赛克图片<br/>
